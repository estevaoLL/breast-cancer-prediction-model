Predição de Câncer de Mama

Sobre o Projeto:

Este foi um trabalho desenvolvido em grupo na faculdade, onde exploramos o uso de Redes Neurais para auxiliar no diagnóstico de câncer de mama a partir de dados clínicos. O objetivo principal foi aplicar técnicas de pré-processamento de dados e treinar um modelo capaz de distinguir entre tumores benignos e malignos.

O que desenvolvemos:

Análise Exploratória: Realizamos o tratamento e limpeza dos dados, utilizando técnicas estatísticas para identificar as correlações mais relevantes e reduzir a redundância entre as características do tumor.

Modelagem: Implementamos um modelo de Rede Neural utilizando a biblioteca TensorFlow/Keras, buscando um equilíbrio entre precisão e capacidade de generalização.

Aprendizado: Durante o processo, aprofundamos nossos conhecimentos em redes neurais densas e no fluxo de trabalho de um projeto de Machine Learning, desde a análise inicial até a avaliação final.
